# UFC Data Visualization and Analysis

This project focuses on exploratory data analysis and visualization of UFC fight data from 1994 to 2025.

The main objective is to analyze UFC events, fight outcomes, locations, and fighter performance through Python-based data visualization techniques.

## Project Overview

The project examines UFC data using multiple tables related to events, fight details, fighter information, and general fight records.

The analysis includes visualizations that help understand the historical development of UFC, the geographical distribution of events, fight result patterns, and performance-related variables.

## Dataset

The project uses a publicly available UFC dataset covering fights and events between 1994 and 2025.

The dataset includes information such as:

* UFC fight records
* Event dates and locations
* Fight outcomes
* Fight methods
* Round information
* Fighter details
* Performance statistics

The dataset is not included in this repository because it is accessed externally.

## Analysis Topics

The project includes:

* UFC event distribution by location
* UFC event distribution by country
* Fight outcome and method analysis
* Fighter performance analysis
* Exploratory data analysis
* Data visualization with Python
* Interactive and static visualizations
